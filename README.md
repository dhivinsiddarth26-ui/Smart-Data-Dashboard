📊 Smart Data Dashboard

A Streamlit-based dashboard that allows users to upload CSV datasets and instantly generate visualizations, statistics, and automated insights.

Features

- CSV file upload
- Dataset preview
- Statistical summary
- Numeric column selection
- Interactive line chart visualization
- Automatic insight generation
- Trend analysis (Increasing, Decreasing, Stable)

Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib

Installation

pip install streamlit pandas matplotlib

Run the Application

python -m streamlit run smart_dashboard.py

How It Works

1. Upload a CSV file.
2. View dataset preview.
3. Analyze numeric columns.
4. Generate charts automatically.
5. Receive smart insights and trend detection.

Future Improvements

- Multiple chart types
- Correlation analysis
- AI-generated insights
- Downloadable reports
